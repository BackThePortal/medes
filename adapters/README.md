# Medes Adapters

Available adapters:
- Curlpp
- D++

## D++

Support for both coroutine-based requests (class `dpp_coro_request_protocol`) and classic callback-based requests 
(class `dpp_request_protocol`).

If you're using [D++ Structures](https://github.com/BackThePortal/dpp-structures), use its request protocol classes, 
as they integrate better with the rest of your code.
