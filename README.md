# Medes — The Modern C++ API Client Library
Medes is a **modern C++ library for building an HTTP API client**. Using metaprogramming, Medes
provides a simple yet powerful API for making **type-safe HTTP requests**. Make your code expressive while knowing it'll 
work before even compiling.

Medes is used as a wrapper over your favourite HTTP requests library; it is HTTP client-agnostic and can be 
adapted to any library.

This library is a good alternative to OpenAPI for **small APIs** or if you don't want to use a code generator.
